# nou.js
Neural network library made from scratch (does not depend on external libraries).

Supports the following layers:
- Fully connected
- RelU
- Leaky RelU
- Tanh
- Sigmoid
- Softmax
- Convolutional
- Global average pooling
- Max pooling
- Average pooling
